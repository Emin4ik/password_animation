Getting start

First: include JQUERY then Bootstrap 3 frameworks

Can get from here: JQUERY framework

or copy from here:
<script src="jquery-3.3.1.min.js"></script>

Can get from here: Bootstrap 3 framework

or copy from here:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" integrity=" sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous">

Second: include my CSS and JS files

<script type="text/javascript" src="js/password_failure_main.js"></script>
<link rel="stylesheet" type="text/css" href="css/password_failure_main.css">

Third: now all you need is just add any class from list below to input and class="btn_animate" to your button

Example of using animations

 <input type="password" class="form-control input_break">
<button class="btn_animate">Enter</button>
